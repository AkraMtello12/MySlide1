# MySlide Agency Website

A high-end presentation design agency website built with React, Tailwind CSS, and Firebase.

## Features

- **Dynamic Content**: Quotes, Resources, and Portfolio managed via Firebase.
- **Admin Dashboard**: Secure area to update site content.
- **RTL Support**: Fully optimized for Arabic language.
- **Responsive Design**: Looks great on all devices.

## Tech Stack

- React 18
- Tailwind CSS
- Firebase (Firestore)
- Vite
- Framer Motion

## Setup

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.

## Deployment

Ready to deploy on Vercel. Ensure you add your Firebase configuration variables if you decide to use environment variables in the future.
